任务¶

Predicting Student Admissions with Neural Networks in Keras¶

In this notebook, we predict student admissions to graduate school at UCLA based on 3 pieces of data:

  • GRE Scores (Test)
  • GPA Scores (Grades)
  • Class rank (1-4) 4档排名

数据 The dataset originally came from here: http://www.ats.ucla.edu/

工具 Keras

  • 深度学习的计算框架
  • 框架: 把一些复杂且通用的流程攒一起完成了，方便直接调用，提高效率

Plotting the data 通过画出来（可视化）¶

# Importing matplotlib
# 引入画图工具库
import matplotlib.pyplot as plt

# Function to help us plot
# 定义画图功能
# 功能（函数）：方便反复调用
# 给函数（plot_points）输入数据（data），画出（散点）图
def plot_points(data):
    # 取每组输入数据的前两维
    X = np.array(data[["gre","gpa"]])   # X: [[660, 3.67], ...]
    # 输出
    y = np.array(data["admit"])   # y: [0, 1, 1 ...]
    
    # 分开被录取的和被拒的
    admitted = X[y==1]   # if admitted, y is 1
    rejected = X[y==0]
    
    # 利用引入的画图工具：画散点图
    # 输入参数：成组的第一维的数，和成组的第二维的数，...
    plt.scatter([s[0] for s in rejected], [s[1] for s in rejected], s = 25, color = 'red', edgecolor = 'k')
    plt.scatter([s[0] for s in admitted], [s[1] for s in admitted], s = 25, color = 'cyan', edgecolor = 'k')
    
    # 坐标名称
    plt.xlabel('Test (GRE)')
    plt.ylabel('Grades (GPA)')
    
# Plotting the points
# 调用上面定义的功能
plot_points(data)   # 画
plt.show()   # 显示出来

3. 预处理数据 Preprocess data¶

预（先）：相对后面的机器学习来说

3.1 One-hot encoding the rank 对排名进行“一位有效”编码¶

排名：categorical data 属于某一范畴（某一档）的数据

  • 范畴之间并没有数值对应的物理意义，比如说：排名第4档并不是第1档的4倍
  • 那么，只要划归某一范畴即可，例如：第4档，那么表示第4档的那一位数为1即可～“一位有效”

For this, we'll use the get_dummies function in numpy. 有可用的函数可以调用，来编码（转换）

In [4]:
# Make dummy variables for rank
# 对排名进行一位有效编码
one_hot_data = pd.concat([data, pd.get_dummies(data['rank'], prefix='rank')], axis=1)

# Drop the previous rank column
# 原来的数值1～4的排名：不要了
one_hot_data = one_hot_data.drop('rank', axis=1)

3.2 Scaling the data¶

调整数据的数值范围（标准化），使其不偏倚、易计算¶

The next step is to scale the data. We notice that the range for grades is 1.0-4.0, whereas the range for test scores is roughly 200-800, which is much larger. This means our data is skewed, and that makes it hard for a neural network to handle. Let's fit our two features into a range of 0-1, by dividing the grades by 4.0, and the test score by 800.

3.4 Splitting the data into features and targets (labels)¶

分开输入（特征）、输出（录取与否的分类：目标值、标签）¶

Also, in Keras, we need to one-hot encode the output. We'll do this with the to_categorical function.

对输出也要进行一位有效编码，“被录取”和“被拒”就是两个范畴，根据录取与否对应地转换记数

4. Defining the model architecture¶

定义模型¶

Here's where we use Keras to build our neural network.

这就是Keras的方便之处

In [13]:
# 引入所需的工具
import numpy as np # 矩阵计算
from keras.models import Sequential # 顺序计算的模型
from keras.layers.core import Dense, Activation # 计算层：全连接，（非线性）激活
from keras.layers.core import Dropout # 为避免训练中过拟合，随机丢掉一些计算单元（神经元）：糙点，宏观点，不钻牛角尖
from keras.optimizers import SGD # 随机梯度下降：优化算法，下山的办法～我们给一小组一小组的数据，也就是给错误山，依法自动下山
from keras.utils import np_utils # 一些辅助工具

# Building the model
# 建小脑袋：结构
model = Sequential()
model.add(Dense(128, activation='relu', input_shape=(6,)))   # 第一层：128个神经元
model.add(Dropout(.2))   # 随机丢百分之多少的神经元，当然了，这些计算单元的过程计算结果就是0了
model.add(Dense(64, activation='relu'))   # 第二层：64个神经元
model.add(Dropout(.1))   # 随机丢百分之多少的神经元
model.add(Dense(2, activation='softmax'))   # 第三层：也就最后一层，用归一化指数层算出录取与否的概率

# Compiling the model
# 编译模型：定义损失函数（山多高），指定优化算法（训练方法）和评价指标（分类准确率）
model.compile(loss='categorical_crossentropy', optimizer='SGD', metrics=['accuracy'])

# 回看模型
model.summary()

8. Take-away and 思考¶

步骤：

  1. 加载数据
  2. 观察数据
  3. 预处理数据
    • 对排名进行“一位有效”编码
    • 调整数据的数值范围
    • 分开训练集、测试集
    • 分开输入、输出
  4. 定义模型（神经网络）
  5. 训练模型
  6. 评价训练结果
  7. 使用模型进行预测
